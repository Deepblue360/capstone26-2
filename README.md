1. 프로젝트 소개

뉴스 기사의 자극적인 인용구(따옴표)를 드래그하여, 실제 발언이 담긴 원본 영상의 정확한 타임라인(시·분·초)을 즉시 찾아내 재생하는 크롬 익스텐션 및 맥락 연결 서비스입니다.

포털 중심 뉴스 생태계의 자극적인 따옴표 보도와 맥락 왜곡(악마의 편집) 문제를 해결하기 위해 기획되었습니다. AI가 참/거짓을 일방적으로 판결하는 심판자 역할을 하는 대신, 독자가 전후 맥락을 직접 확인할 수 있는 객관적인 팩트의 경로(원본 풀영상의 타임라인)를 즉시 제공함으로써 독자 스스로 주체적인 판단을 내릴 수 있도록 돕습니다.

2. 동작 과정

사용자가 뉴스 기사 속 문장을 마우스로 드래그하는 순간부터 원본 영상이 재생되기까지, 다음의 5가지 단계를 거쳐 스마트하게 작동합니다.

① 뉴스 기사 정보 자동으로 읽기

사용자가 뉴스 본문에서 궁금한 따옴표 문장을 마우스로 드래그하고 익스텐션을 실행하면, 서비스가 해당 문장과 기사 정보(제목, 날짜, 언론사 등)를 자동으로 가져옵니다.

② 발언이 나온 진짜 출처(행사) 알아내기

가져온 기사 정보를 바탕으로, AI가 "이 발언이 실제로 행해진 원래 행사(예: 특정 국회 청문회, 기자회견, 방송 프로그램 등)가 무엇인지" 스스로 분석하고 알아냅니다.

③ 유튜브에서 신뢰할 수 있는 원본 영상 찾기

알아낸 행사 이름을 바탕으로 유튜브에서 관련 영상들을 검색합니다. 이때 단순히 조회수가 높은 영상이 아니라, 짜깁기 없는 공식 방송국의 긴 풀영상(Full Video)을 우선적으로 골라냅니다.

④ 자막과 문장을 비교해 정확한 시간 찾아내기

찾아낸 원본 영상들의 자막 데이터와 사용자가 드래그했던 문장을 정밀하게 비교합니다. AI가 글자 그대로의 비교를 넘어 문맥과 의미를 파악하여, 내가 드래그한 발언이 영상의 몇 분 몇 초에 정확히 나오는지 찾아냅니다.

⑤ 화면에 영상 바로 띄우기

찾아낸 정확한 시간 정보(타임스탬프)를 뉴스 화면으로 다시 보내줍니다. 사용자는 다른 사이트로 이동할 필요 없이, 읽던 뉴스 기사 창 위에 바로 떠오르는 영상 플레이어를 통해 실제 발언의 전후 맥락을 즉시 확인할 수 있습니다.
