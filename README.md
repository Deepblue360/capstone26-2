



# 🔍 TruthSync
> **"단 한 번의 클릭으로, 기사 속 따옴표에서 원본 맥락 영상까지."**

---

## 📌 문제 정의 및 프로젝트 소개
<img width="460" height="297" alt="image" src="https://github.com/user-attachments/assets/fbc4a4e4-3bcf-4a94-a40d-ad8d23f40386" />
<img width="335" height="71" alt="image" src="https://github.com/user-attachments/assets/86e2a653-eb01-4582-bebb-9f3b6d0989ff" />


* 우리는 뉴스 기사에서 수많은 인용구(따옴표)를 많이 볼수있고, 기사 내 인용문은 계속해서 늘어나고 있는 추세입니다.
* 하지만 인용구를 사용해 기사를 작성할 때, 발화자의 전체 발언 내용이 아닌 일부 내용만을 발췌하여 인용하는 것이 대부분입니다.
* 이런 편집 과정에서, 발화자의 본래 맥락이 기존 의도와는 다르게 왜곡되거나 변질되는 문제가 발생합니다.
* 심지어 특정 프레임을 씌우기 위해 이러한 맥락 왜곡을 교묘하게 의도하는 매우 악질적인 경우도 존재합니다.


이러한 문제를 해결하기위해 TruthSync는 **뉴스 기사의 인용구를 원본 영상의 정확한 타임라인을 찾아내 재생하는** 기능을 통해 독자에게 원본 맥락을 판단할수있도록 도와줍니다.

---

## 🔍 동작 과정

### 1️⃣ 뉴스 기사 정보 자동으로 읽기
* 사용자가 뉴스 본문에서 궁금한 따옴표 문장을 마우스로 드래그하고 익스텐션을 실행하면, 서비스가 해당 문장과 기사 정보(제목, 날짜, 언론사 등)를 수집

### 2️⃣ 발언이 나온 출처 알아내기
* 가져온 기사 정보를 바탕으로, claud api를 통해 이 발언이 실제로 행해진 원래의 행사(예: 특정 국회 청문회, 기자회견, 방송 프로그램 등)가 무엇인지 스스로 분석하고 추론

### 3️⃣ 유튜브에서 신뢰할 수 있는 원본 영상 찾기
* 알아낸 행사 이름을 바탕으로 유튜브에서 관련 영상을 검색하고, 가장 정확성이 높은 영상 순으로 순위를 매김

### 4️⃣ 자막과 문장을 비교해 정확한 시간 찾아내기
* 원본 영상의 자막 데이터와 사용자가 드래그했던 문장을 정밀하게 비교하며,  문맥과 의미를 파악해 발언이 나오는 정확한 타임라인을 검출

### 5️⃣ 해당 영상 타임라인 부분 실행
* 사용자가 타임라인 버튼 클릭시 자동으로 해당 시점의 영상이 뜨고 재생됨

## ✔️ 결론
* **TruthSync는 자극적인 따옴표 보도에 휘둘리지 않고 독자가 직접 원본의 맥락을 확인하여 정보의 판단의 주체성을 회복하도록 돕습니다.**
* **일부 발췌나 악의적인 짜깁기로 인한 맥락 왜곡을 방지하고, 객관적인 내용 전달의 새로운 대안을 제시합니다.**
* **궁극적으로는 왜곡된 뉴스 소비 환경을 바로잡고, 건강한 미디어 생태계를 만들어 나가는 데 기여하고자 합니다.**
